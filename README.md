#Mazerunner

##Command Line Arguments
- -m loads a specified maze file.  Default is "maze.dat"
- -a loads an algoithm.  Default is "tremaux.so"

##Controls
- w Move forward
- a Turns mouse left
- d Turns mouse right
- R Resets mouse 
- r Right wall hugger
- l Left wall hugger
- g Toggle ghost mode
- i Toggle invincibility
- space Step algorithm
