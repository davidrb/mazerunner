#Mazerunner

##Command Line Arguments
- -m loads a specified maze file.  Default is "maze.dat"
- -a loads an algoithm.  Default is "tremaux.so"

##Controls
- w moves the mouse
- a turns mouse left
- d turns mouse right
- R resets mouse 
- r right wall hugger
- l left wall hugger
- g toggle ghost mode
- i toggle invincibility
- space runs the algorithm
